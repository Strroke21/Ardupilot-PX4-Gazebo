# ArduPilot–PX4 Gazebo

This repository provides simulation resources for **ArduPilot** and **PX4** using **Gazebo Garden**. It contains models, worlds, plugins, and configuration files for UAV simulation with **ROS 2 Humble**.

## Features

- Gazebo Garden support
- ArduPilot and PX4 vehicle models
- Custom simulation worlds
- ROS 2 Humble compatible
- Sensor and plugin integration

## Requirements

- Ubuntu 22.04
- ROS 2 Humble
- Gazebo Garden
- ArduPilot and/or PX4

## Clone

```bash
git clone https://github.com/Strroke21/Ardupilot-PX4-Gazebo.git
```
